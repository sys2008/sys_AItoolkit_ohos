# sys_AItoolkit_ohos

> 由宋阳烁独立开发的鸿蒙原生本地大模型工具箱

一款专为 HarmonyOS 打造的纯本地化 AI 与数学计算工具集。无需联网，数据不出端，在提供强大本地大模型推理能力的同时，兼顾了专业级别的数学计算需求。

## ✨ 核心功能

### 🧮 精准科学计算器
彻底告别“近似小数”时代。内置专业数学解析引擎，支持**保留根号（如 $\sqrt{2}$）**和**保留分数（如 $\frac{1}{3}$）**的精确计算逻辑，杜绝浮点数精度丢失，还原最真实的数学结果。

### 📐 智能方程求解
支持各类常见方程的输入与自动解析（涵盖一元一次方程、一元二次方程等），直接输出准确解，是学习与工作中的得力辅助工具。

### 🦙 llama.cpp 本地大模型运行
深度集成业界主流的 llama.cpp 推理框架，支持在鸿蒙设备上直接加载并运行 GGUF 格式的开源大语言模型（如 Qwen、Llama 等），实现完全断网情况下的智能对话与文本生成。

### ⚡ MNN 端侧推理加速
接入阿里巴巴开源的 MNN 推理引擎，针对端侧设备底层进行极致优化。为用户提供除 llama.cpp 外的第二种高效本地大模型运行方案，兼顾不同模型的兼容性与运行效率。

### 🛡️ 纯本地隐私护城河
所有计算与 AI 推理过程 100% 在设备本地完成。不收集、不上传、不依赖云端接口，将数据隐私的主动权彻底交还给用户。

### 🎨 鸿蒙原生体验
基于 HarmonyOS 原生框架开发，从底层到 UI 全面适配鸿蒙系统设计规范，交互流畅，体验纯净。
